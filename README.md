# diqibadao/hermes-skills

Hermes Agent skills.

## Install

```
hermes skills tap add diqibadao/hermes-skills
hermes skills search
hermes skills install system-health
hermes skills install user-identity
```

## Skills

### system-health
System health monitoring. 6 layers, auto-remediation, daily reports.

### user-identity
Cross-platform user identity. Shared across Hermes/OpenClaw/Cursor.
